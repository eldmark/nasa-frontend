# Reflexión

Durante este proyecto aprendí a usar GitHub Actions para automatizar el build y el despliegue de una aplicación frontend en GitHub Pages.

También aprendí cómo llevar la inicialización de una base de datos a producción, aplicando el esquema SQL en una base remota y conectando el backend a Render mediante una URL de conexión externa o interna según el servicio.

Otro punto importante fue el uso de middlewares en el backend para validar datos antes de llegar a los controladores y mantener una arquitectura más ordenada.

Por último, reforcé buenas prácticas para Single Page Applications, como usar routing por hash, separar la lógica por módulos y manejar la configuración del backend con variables de entorno.
